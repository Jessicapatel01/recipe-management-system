# recipe-management-system
A Java-based Recipe Management System built from scratch using Object-Oriented Programming and Collections.

A Java-based console application for managing recipes, built from scratch to strengthen my understanding of Object-Oriented Programming (OOP), Java Collections, and software development fundamentals.

> 🚧 This project is under active development and new features are being added regularly.

---

Features:

 Implemented features:

- ➕ Add a new recipe
- 📖 View all recipes
- 🔍 Search recipes by name
- 🗑️ Delete recipes
- 🔄 Menu-driven console interface
- 📦 Object-Oriented Design using Java classes
- 📚 Recipe storage using `ArrayList`

 In Progress->

 Update/Edit recipes

 Planned Features:

- 💾 Save recipes permanently using File Handling
- 📂 Recipe categories
- 🥗 Nutrition tracking
- 🥕 Search by ingredients
- ⭐ Favorite recipes
- 🍽️ Servings and cooking time
- 🌐 Recipe API integration
- 🖼️ Recipe images

- 🌍 Web version using Spring Boot

## 🚀 How to Run

1. Clone the repository

```
git clone: https://github.com/Jessicapatel01/recipe-management-system.git
```

2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse).

3. Compile and run `Main.java`.
